Minecraft Java版（Windows）下载、Mod安装与联机保姆级教程

|总字数:987|阅读时长:3分钟|浏览量:

前情提要

本文旨在帮助Minecraft新手下载并游玩Minecraft Java，若内容出现错误，欢迎指出。6383412@qq.com
!!! 请注意，以下所有教程皆以Windows为系统环境。

本文讲解内容一览：

前置准备（重要）

下载java（以jdk17为演示）

在Minecraft Java版里，顾名思义，Java环境是必要的。

首先，这是一张Minecraft Java版与Java版本对应关系表

Minecraft版本 推荐Java版本 说明
1.0 ～ 1.5.2 Java 6 / 7 超老版本（基本不用了）
1.6 ～ 1.8.9 Java 8 经典老版本 / 很多老模组
1.9 ～ 1.12.2 Java 8 最经典模组生态（Forge老整合包）
1.13 ～ 1.16.5 Java 8 / 11 过渡时期版本
1.17 ～ 1.20.4 Java 17 现代主流模组版本
1.20.5 ～ 1.21+ Java 21 最新版本

如果你是从后面来这里的，点击此链接即可快速返回。

现以JAVA17作为教程演示。

下载JDK17安装包

由于国内网络环境很难直接访问到甲骨文官网，所以这里我提供了JDK8、JDK11、JDK17、JDK21的安装包。点击下列对应JAVA版本即可直接下载安装包。
(Oracle官网下载链接)

jdk-8u341-windows-x64.exe
jdk-11.0.18_windows-x64_bin.exe
jdk-17.0.14_windows-x64_bin.msi
jdk-21_windows-x64_bin.msi

下载后打开安装包

选择一个安装路径（记住这个路径），并点击NEXT

点击Close

因为只是玩MC，所以不需要配置环境变量
到这里，JAVA就算是安装好了。

接下来，即是正文。

1、游戏启动器PCL

如何下载PCL

打开PCL创作者龙腾猫跃的爱发电主页。

–>一键直达<–

点击页面左侧的链接

随便点一个下载PCL本体(建议选择备用下载链接，因为备用下载链接下载不限速)

将链接复制进网址栏(如图所示),然后按下Enter访问。

在弹出方框中输入pcl2

点击图中标记的链接，下载pcl2

下载好后，将Plain Craft Launcher 2.exe解压到空文件夹中

双击打开Plain Craft Launcher 2.exe

到这里，PCL本体就下载完成了

登录环节

如果是离线版本登录游戏(没花钱买正版账号的玩家)

在此界面的游戏用户名文字框中，填写您的游戏名称。

注意！游戏名称最好只有英文``数字以及下划线“_”，否则游戏可能会报错！！！！！

填写完名称后，即可直接跳转至此章节进行下一步

如果您是有正版账号的玩家

在此界面的左上角点击正版

点击登录按钮

在自动打开的网页中将登录代码复制到文字框中，点击允许访问

点击接受按钮

当出现玩家头像和名字时，即登录成功。

2、如何在PCL中下载Minecraft游戏本体

2.1、游戏各版本区别

在Minecraft中，玩家可以自行选择各个发行版进行游玩。
那么，在这些琳琅满目的游戏版本中，该如何选择适合自己的呢？

首先。我们来了解一下这些游戏版本的区别。
(其实在前面已经有一张表格描述过了。点这里返回观看)

虽然，1.7.10 1.12.2之类的MC版本都是公认的“神中神”，
但是，目前，由于一些不可抗力的因素，本人在这里并不推荐游玩这些低版本MC。
（除非你的电脑配置实在是很低很低很低，带不动高版本的情况下才选择这些低版本）
所以，本人推荐游玩的版本有1.16.5(下界更新) 1.18.2(洞穴更新)以及最新版本(最新体验)。
接下来的教程会以1.18.2为示例版本进行教程制作。
