MC“元素周期表”模组：Fabric 环境下的性能优化方案

一、什么是“元素周期表”模组？

“元素周期表”是 Minecraft 社区对一系列以化学元素命名的优化模组的统称。

这个称呼的由来很简单：Sodium（钠） 是第一个打出名气的优化模组，对游戏性能的提升非常显著，帧数表现极其稳定，因此知名度极高 参考来源: Modrinth。随后越来越多的优化模组效仿 Sodium 的命名方式，纷纷以元素周期表中的元素命名，社区便形象地将它们统称为“元素周期表” 。

注意：“元素周期表”并非一个整合包或单一模组，而是一个模组集合的统称。你需要根据需求分别下载对应的模组文件。

二、Sodium（钠）

定位：渲染引擎优化模组

Sodium 是一个适用于 Minecraft 1.16+ 客户端的免费、强大的渲染引擎优化模组 参考来源: Modrinth。它在修复一些图形问题的同时，大幅提高了渲染性能 参考来源: Github。

核心作用

Sodium 完全重写了游戏的原版渲染引擎，使其速度更快、效率更高 参考来源: Github。根据官方数据，FPS 通常可提高 300% 或更多 参考来源: Github。同时它还能减少微卡顿（micro-stutter），让游戏运行更加丝滑。

关键特性

  • 不改变原版画面：Sodium 的优化不会影响游戏的视觉效果，让你体验到原汁原味的方块游戏。
  • 广泛的模组兼容性：与 Fabric 模组生态系统具有广泛的兼容性。
  • 即装即用：默认情况下，Sodium 会启用系统支持的所有优化，一般不需要手动配置。

使用前提

Sodium 是一个 Fabric 模组，需要 Fabric Loader 作为前置 参考来源: Github。安装 Fabric Loader 后，将 Sodium 的 JAR 文件放入 mods 文件夹即可。

兼容性警告：Sodium 与 OptiFine（高清修复）不兼容 参考来源: Github。如果你之前在使用 OptiFine，需要先将其移除。

版本说明

  • Sodium 0.9.x 已支持 Minecraft 26.2，并包含对 Vulkan 的早期支持 参考来源: Modrinth。
  • Sodium 0.6.0 及以上版本不需要也不兼容 Indium（铟） 这个兼容性模组 参考来源: Github。

三、辅助成员：Lithium（锂）

定位：游戏逻辑运算优化模组

Lithium 是一个自由及开源的优化模组，与 Sodium 专注于渲染不同，Lithium 致力于优化游戏的各种逻辑系统 参考来源: Modrinth。

核心作用

Lithium 在不修改原版机制的前提下，改善多种原版机制的计算方法 参考来源: Github，包括：

  • 物理系统优化：减少碰撞箱的分辨率复杂度，优化实体碰撞检测。
  • 生物 AI 优化：使用基于事件的系统减少 CPU 占用，生物在不同 AI 任务间的切换更快。
  • 世界生成优化：移除不必要的计算，显著降低生成新区块时的性能开销。
  • 区块加载优化：玩家探索世界时 TPS 下降更少，加载时间缩短。
  • 刷怪塔优化：优化实体碰撞处理，显著降低刷怪塔对服务器 TPS 的影响。

使用说明

Lithium 分为 Fabric 和 NeoForge 两个版本 参考来源: Modrinth。可单独安装至客户端或服务器任意一端——服务端安装时客户端无需安装，反之亦然。

注意：Lithium 不再更新 Forge 版本，现仅支持 Fabric 和 NeoForge 参考来源: Github。

Sodium 官方文档明确指出：建议同时安装 Lithium 来优化游戏的其他部分 参考来源: Github。

四、辅助成员：Phosphor（磷）

定位：光照引擎优化模组

Phosphor 是一个致力于优化 Minecraft 中优化最差的部分——光照引擎的模组 参考来源: Modrinth。

核心作用

  • 大幅降低区块生成时间：使用 Phosphor 后，在某些维度下游戏生成区块所用时间可减半，生成整个世界的时间也将大幅降低 参考来源: Github。
  • 修复光照错误：修复了一些长期存在的光照引擎 bug。
  • 服务端/客户端通用：支持客户端和服务器，服务端安装时客户端无需安装。
  • 不改变原版特性：Phosphor 不会改变原版游戏的特性，且支持绝大多数地形模组。

重要状态说明

Phosphor 已停止维护 参考来源: Github。自 Minecraft 1.20 版本以来，游戏的光照引擎已经有了显著改进，Phosphor 的大部分补丁已不再适用或有用。

  • 如果你使用 1.20 及以上版本：不需要安装 Phosphor 参考来源: Modrinth。
  • 如果你使用 1.20 以下版本：Phosphor 仍然有效，可以继续使用。

兼容性：Phosphor 不兼容 Starlight（星光） ，因为二者都重写了光照引擎。请二选一 参考来源: Github。

五、元素周期表完整组合安装指南

适用环境

  • Minecraft 版本：1.16.5 或更高版本
  • 模组加载器：Fabric 或 NeoForge
  • Java 运行环境：建议 Java 17 或更高版本

推荐组合（按优先级）

模组 必要性 说明
Fabric Loader 必装 模组加载器基础
Sodium（钠） 必装 渲染优化核心
Lithium（锂） 推荐 逻辑运算优化
Phosphor（磷） 1.20以下推荐 光照优化（1.20+ 不需要）

六、总结

方案 适用场景 推荐组合
纯性能优化（Fabric） 不玩光影，只求流畅 Sodium + Lithium +（1.20以下加 Phosphor） 参考来源: Modrinth
纯性能优化（Forge） Forge 环境下的替代 Embeddium + Canary（或 Starlight） 参考来源: Modrinth

这套“元素周期表”组合，是目前 Fabric 环境下性能最优、兼容性最好、完全开源的优化方案，能显著提升游戏帧率并减少卡顿，值得每一位注重流畅度的玩家尝试。
